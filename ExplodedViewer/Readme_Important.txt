README

To make this script work:
1.) Open Maya
2.) Launch script editor. Create a Python tab inside script editor
3.) Open the script file "ExpodedViewer_Production.py" and copy it. Then paste it into the Python script tab
4.) Run the script, and be sure to check out the Help tab inside the Exploded Viewer editor

See the script in action here: https://www.artstation.com/artwork/xJab4R
